## 🎥 Production Tips

### Recording Setup

1. **Screen Recording**
   - Use OBS Studio or QuickTime
   - 1920x1080 resolution minimum
   - 60fps for smooth animations

2. **Audio**
   - Use quality microphone for voice commands
   - Record narration separately for clarity
   - Add subtle background music (royalty-free)

3. **Visuals**
   - Show Ghost Trace sidebar throughout
   - Use split-screen for voice + UI
   - Highlight key UI elements with circles/arrows

### Editing Checklist

- ✅ Start with voice interaction in first 10 seconds
- ✅ Show "Gemini Live API" badge prominently
- ✅ Display "Hosted on Google Cloud" throughout
- ✅ Include Ghost Trace in every demo
- ✅ Show real-time latency metrics
- ✅ Add captions for accessibility
- ✅ Keep total length under 5 minutes
- ✅ End with clear call-to-action

### Key Messages to Emphasize

1. **"Live" is the hero** - Voice interaction first
2. **Google Cloud hosting** - Show Cloud Run dashboard
3. **Grounded intelligence** - Citations and sources
4. **Production-ready** - Real metrics, not prototypes
5. **Transparent AI** - Ghost Trace throughout
